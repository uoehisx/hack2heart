제안 콘셉트

  1. Code Brackets Heart (현재 시안 확장)

    • 핵심 심볼: < > 안에 하트(또는 하트 안에 </> 작은 코드 아이콘)

    • 색상: 진한 개발자 느낌의 다크(차콜/네이비) + 포인트 레드 (#FF3B30 근접)

    • 활용: 아이콘(하트만), 풀로고(하트 + 텍스트) 두 버전.

  2. Binary Pulse Heart

    • 하트 윤곽선 일부가 1011001 같은 비트 패턴으로 픽셀화되어 사라지거나 나타나는 효과.

    • 포인트 색: 네온 핑크 or 사이버 민트 + 다크 배경.

    • 앱 아이콘에서 기술감 & 에너지.

  3. Terminal Cursor Love

    • 검은 원(or 둥근 사각) 안에 _ 커서 뒤에 love: 처럼 프롬프트 형태, 커서 옆에 작은 하트.

    • 타이포: 모노스페이스(IBM Plex Mono, JetBrains Mono 느낌) + “Hack2Heart”는 Bold, tagline은 Regular.

  4. Connected Nodes

    • 작은 원(노드)들이 그래프(네트워크) 형태로 연결되어 하트 형상을 이룸.

    • 색: 그라데이션 (보랏빛 -> 핑크).

    • “개발자들의 연결” 메시지 강조.

  5. Merge Conflict Resolved

    • Git 브랜치 두 개가 합쳐져 하트 모양을 이루는 심볼 (두 곡선이 하트 상단 양쪽).

    • 작은 체크(✔) 또는 머지 아이콘이 하단에.

    • 협업·매칭 의미.

  6. Angle Bracket + Number 2 Integration

    • <2를 좌측 상단, />를 우측 하단 배치해 전체가 하트 윤곽을 암시.

    • “2”를 ‘to’(연결) 상징으로 시각화.

  7. Dark Mode / Light Mode 대응형

    • Light: 다크 그레이 텍스트 + 레드 하트.

    • Dark: 화이트 텍스트 + 살짝 밝은 코랄/핑크 하트.

    • 자동 전환 가능한 두 팔레트 세트.
